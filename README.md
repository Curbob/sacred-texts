# ✨ The Sacred Texts of Code ✨

**Ancient programming wisdom preserved in a beautiful interactive manuscript**

## 🌐 Live Site
**https://curbob.github.io/sacred-texts** *(will be live after deployment)*

## 📜 About

The Sacred Texts of Code is an illuminated manuscript containing ancient programming wisdom, presented through a modern dual-page reading interface. Experience the convergence of timeless knowledge and cutting-edge web technology.

### ✨ Features
- 📖 **Dual-Page Reading Mode** - Toggle between single and dual-page views
- ⛶ **Fullscreen Experience** - Immersive manuscript reading
- 📱 **Mobile Optimized** - Touch-friendly navigation on all devices
- 🔒 **Protected Content** - Right-click protection for sacred pages
- 🎨 **Medieval Theme** - Beautiful golden illuminated styling

## 🛠️ Built With

- **[ImageFlipbook v1.1.1](https://github.com/Curbob/image-flipbook)** - Interactive flipbook library
- **Security hardened** with HTTPS enforcement and content protection
- **Zero dependencies** - Pure JavaScript implementation
- **Modern web standards** - Progressive enhancement

## 📚 Manuscript Contents

The Sacred Texts contain 68 illuminated pages including:

1. **Frontispiece** (Pages 1-8) - Hours of AI, Saints, Great Litany
2. **Saints & Hours** (Pages 9-25) - Remaining Hours and More Saints  
3. **Sacred Protocols** (Pages 26-49) - Litanies, Daily Offices, Wisdom
4. **Gospels & Wisdom** (Pages 50-67) - Literature, Gospels, Calendars, Sacred Geometry
5. **Colophon** (Page 68) - Manuscript completion

## 📖 Reading Instructions

### Desktop
- **Single/Dual Toggle**: Use the "1 Page" / "2 Pages" buttons to switch views
- **Navigation**: Arrow keys or click the Previous/Next buttons
- **Fullscreen**: Click any page or the fullscreen button (⛶)
- **Chapters**: Jump to sections using chapter buttons

### Mobile
- **Touch Navigation**: Swipe or tap navigation buttons
- **Zoom**: Pinch to zoom on individual pages
- **Auto-Mode**: Automatically optimizes for your screen size
